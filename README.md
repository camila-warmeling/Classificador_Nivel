# 🦸‍♂️ **Classificador de Nível de Herói**

Programa em Python que classifica o nível de um herói com base na quantidade de XP (experiência) informada pelo usuário.
O usuário pode cadastrar vários heróis e, ao final, o sistema informa o nível correspondente de cada um.


📚**Conceitos Utilizados:** 

-Estruturas de Repetição
-Estruturas Condicionais
-Tratamento de Exceções
-Listas em Python
-Entrada e Saída de Dados


📚 **Sistema de Classificação:**

🏆 XP do Herói	Nível  
até 1000	🪨 Ferro  
1001 - 2000 🥉 Bronze  
2001 - 5000	🥈 Prata  
5001 - 7000	🥇 Ouro  
7001 - 8000	💎 Platina  
8001 - 9000	⚡ Ascendente  
9001 - 10000	🔥 Imortal  
acima de 10000	🌟 Radiante


▶️ **Como Executar:**

1️⃣ Clone o repositório:
	git clone https://github.com/seu-usuario/seu-repositorio.git

2️⃣ Entre na pasta:
	cd nome-do-repositorio

3️⃣ Execute o programa:
	python nome_do_arquivo.py


💻 **Exemplo de Execução:**
	
  Digite o nome do herói (ou enter para encerrar): Thor
	Digite a quantidade de xp do herói: 3500

  Digite o nome do herói (ou enter para encerrar): Hulk
  Digite a quantidade de xp do herói: 12000

  O Herói Thor está no nível Prata
  O Herói Hulk está no nível Radiante
